### 1. 主标题图（必需）

**Figma节点**: `1:4852` (组 1 1)
**文件名**: `hero-title.png`
**保存路径**: `public/images/home/hero-title.png`

**格式**: PNG（带透明背景）

#### 导出步骤：
1. 在Figma中选中节点 `1:4852` (组 1 1)
2. 右侧导出面板选择 PNG 格式
3. 导出设置：
   - 分辨率：2x（为了保证清晰度）
   - 背景：透明
4. 导出后重命名为 `hero-title.png`
5. 放置到 `public/images/home/` 目录

## ⚡ 快速导出命令（如果使用Figma CLI）

```bash
# 主标题图
figma-export --node-id=1:4852 --format=png --scale=2 --output=public/images/home/hero-title.png

# 福州大学Logo  
figma-export --node-id=I1:4859;1:6537 --format=png --scale=2 --output=public/images/home/fzu-logo.png
```

## 📝 注意事项

1. 所有PNG图片导出时请确保**透明背景**
2. 建议使用 **2x** 分辨率导出，保证高清显示
3. 导出后使用图片压缩工具（TinyPNG）压缩文件大小
4. 确保文件名**完全匹配**代码中的引用路径
