## 📥 获取完整图片资源

### 方案一：从Figma导出（推荐）

1. 打开Figma设计文件
2. 选中节点 `1:4852` (组 1 1)
3. 右侧面板选择"导出"
4. 设置：
   - 格式：PNG
   - 分辨率：2x
   - 背景：透明
5. 导出并重命名为 `hero-title.png`
6. 放置到 `public/images/home/` 目录


## 📝 技术说明

### 图片加载失败处理
```vue
<img 
  v-if="heroTitleImageExists" 
  src="/images/home/hero-title.png" 
  @error="heroTitleImageExists = false"
/>
<h1 v-else class="home__hero-title-text">
  <!-- 文字版后备 -->
</h1>
```

### 背景图精确定位
```css
.home__hero-bg-image {
  position: absolute;
  top: -19.04%;      /* 按设计稿偏移 */
  height: 126.98%;   /* 按设计稿缩放 */
}
```

### 金色渐变文字效果
```css
background: linear-gradient(180deg, #ffd89a 0%, #ffb347 50%, #d4905f 100%);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```
