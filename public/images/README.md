## 图片规格要求

### 英雄横幅图
- 尺寸：1920x756px
- 格式：JPG
- 质量：高质量
- 内容：党建主题，红色调为主

### 课程封面图
- 尺寸：364x200px
- 格式：JPG
- 质量：中等质量
- 内容：学院建筑或课程相关场景

### 视频缩略图
- 大图：320x212px
- 小图：320x172px
- 格式：JPG
- 质量：中等质量

### Logo和图标
- 格式：PNG（支持透明背景）
- 质量：高质量
- 注意保持清晰度

## 颜色规范

根据设计稿，图片应遵循以下色彩规范：
- 主色：#BC2220（思政红）
- 辅色：#333333（基础黑）
- 背景：#FFF9ED（米黄）
- 文字：#FFFFFF（白色）

## 注意事项

1. 所有图片均需优化，控制文件大小
2. 使用适当的图片压缩工具（如 TinyPNG、ImageOptim）
3. 保持图片长宽比，避免变形
4. 确保图片版权合规
5. 建议使用 WebP 格式以获得更好的压缩效果（需要同时提供 JPG/PNG 作为后备）
