## 🔍 关键技术点

### 1. 自动降级机制
```vue
<!-- 优先使用图片 -->
<img v-if="heroTitleImageExists" src="..." @error="fallback" />
<!-- 图片失败自动切换文字 -->
<h1 v-else>文字版</h1>
```

### 2. 精确定位（按Figma设计）
```css
/* 背景图 */
top: -19.04%;
height: 126.98%;

/* 主标题 */
top: 38px;
left: calc(50% - 30px);
```

### 3. 金色渐变效果
```css
background: linear-gradient(180deg, 
  #ffd89a 0%, 
  #ffb347 50%, 
  #d4905f 100%
);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```
