## 背景样式改动

### 改动前
```css
.excellent-courses::after {
  background: linear-gradient(180deg, #a40a0b 0%, #9c090a 100%);
}
```
- 仅使用纯色渐变背景
- 从深红色渐变到红色

### 改动后
```css
.excellent-courses::after {
  background: url('/images/Frame_1000015361.png') center/cover no-repeat, 
              linear-gradient(180deg, #a40a0b 0%, #9c090a 100%);
}
```
- 使用背景图片 + 渐变的多层背景
- 图片作为主背景
- 渐变作为后备背景

## 技术实现

### 多层背景语法
```css
background: 
  url('/images/Frame_1000015361.png') center/cover no-repeat,  /* 第一层：图片 */
  linear-gradient(180deg, #a40a0b 0%, #9c090a 100%);           /* 第二层：渐变 */
```

### 背景参数说明
- `center/cover` - 居中显示，覆盖整个区域
- `no-repeat` - 不重复
- 渐变作为后备，确保图片加载失败时仍有背景

## 常见问题

### Q1: 图片不显示怎么办？
**检查**：
- 确认图片路径正确
- 查看浏览器控制台是否有 404 错误
- 确认图片文件存在于 `public/images/` 目录

### Q2: 图片显示变形？
**解决**：
使用 `cover` 而不是 `100% 100%`：
```css
background-size: cover; /* 保持比例 */
```

### Q3: 背景图片模糊？
**原因**：图片分辨率不够
**解决**：使用更高分辨率的图片（建议 1920px 以上）

### Q4: 影响页面加载速度？
**解决**：
1. 压缩图片文件大小
2. 使用图片懒加载
3. 转换为 WebP 格式

### Q5: 文字在背景上不清晰？
**解决**：添加半透明叠加层：
```css
background: 
  linear-gradient(rgba(0,0,0,0.3), rgba(0,0,0,0.3)),
  url('/images/Frame_1000015361.png') center/cover;
```
