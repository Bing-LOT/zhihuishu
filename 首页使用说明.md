# 首页使用说明 🏠

## ✅ 已完成功能

您好！福州大学新时代思政教育平台的首页已经按照Figma设计稿完整实现。以下是已完成的功能：

### 🎯 核心功能

1. **英雄横幅轮播** - 3张横幅图自动轮播，支持手动切换
2. **视频展示区** - 背景视频播放 + 视频缩略图轮播
3. **学院案例展示** - 8个课程卡片，2行4列网格布局
4. **导航栏和页脚** - 完整的布局组件

### 📦 组件清单

- ✅ CourseCard（课程卡片组件）
- ✅ VideoCard（视频卡片组件）
- ✅ PortalLayout（门户布局）
- ✅ Home（首页主体）

## 🚀 快速启动

### 方法一：命令行启动

```bash
# 1. 进入项目目录
cd zhihuishu-app

# 2. 安装依赖（首次运行）
npm install

# 3. 启动开发服务器
npm run dev
```

服务器启动后，浏览器会自动打开 http://localhost:5173

### 方法二：使用IDE

如果您使用 VS Code：
1. 打开项目文件夹
2. 按 `F5` 或点击"运行和调试"
3. 选择 "Launch Chrome against localhost"

## ⚙️ 当前使用占位图

由于实际图片资源尚未到位，目前使用在线占位图服务（Unsplash + Placeholder.com）。

**占位图配置文件**: `public/images/placeholders.json`

如需替换为真实图片：
1. 将图片文件放入 `public/images/` 对应目录
2. 参考 `public/images/README.md` 中的尺寸规范
3. 重启开发服务器

## 🎯 交互功能测试

启动后可以测试以下交互：

1. **横幅轮播**
   - 等待5秒，自动切换到下一张
   - 点击左右箭头手动切换
   - 点击底部圆点快速跳转

2. **滚动继续按钮**
   - 点击后页面平滑滚动到视频区

3. **视频缩略图**
   - 点击左右箭头切换视频
   - 当前选中的缩略图会变大

4. **课程卡片**
   - 鼠标悬停：卡片上移 + 阴影
   - 点击卡片：跳转到课程详情页

5. **导航菜单**
   - 当前页面的菜单项会高亮显示

## ⚠️ 可能遇到的问题

### 1. 图片不显示
**解决**: 占位图需要网络连接。如果无法访问在线图片服务，请：
- 检查网络连接
- 或准备实际图片文件

### 2. 视频不播放
**解决**: 背景视频文件 `public/videos/hero-video.mp4` 尚未添加。
临时方案：视频区会显示为黑色背景，不影响其他功能测试。

### 3. 字体显示异常
**解决**: Source Han Sans CN 会自动降级到系统中文字体（微软雅黑、苹方等）。
