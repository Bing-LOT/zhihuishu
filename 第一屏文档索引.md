## 🚀 快速操作指南

### 方案一：最快启动（0配置）
```bash
cd zhihuishu-app
npm run dev
```
访问 http://localhost:5173 即可看到效果（文字版）

### 方案二：添加图片（推荐）
1. 打开 `scripts/create-hero-placeholder.html`
2. 下载图片并重命名为 `hero-title.png`
3. 放到 `public/images/home/` 目录
4. 刷新页面

### 方案三：从Figma导出（最佳）
参考 **首页第一屏资源说明.md** 的详细步骤
