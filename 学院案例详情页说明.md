# 学院案例详情页说明

## 概述

已完成学院案例详情页的开发，包含视频播放、文字讲稿展示和推荐视频列表等功能。

## 功能特性

### 1. 页面布局

- **顶部导航栏**：红色主题导航栏（复用Portal Layout）
- **面包屑导航**：显示当前位置 "首页 >> 学院案例 >> 视频详情"
- **左侧区域**：文字讲稿卡片，展示课程介绍内容
- **右侧区域**：
  - 课程标题和观看次数
  - 视频播放器（支持播放/暂停、进度控制、全屏）
  - 教工信息
- **底部区域**：其他教学岗教工视频推荐（4个卡片）
- **页脚**：福州大学相关信息

### 2. 视频播放器功能

- ✅ 自定义视频控制栏
- ✅ 播放/暂停切换
- ✅ 进度条拖动
- ✅ 时间显示（当前时间/总时长）
- ✅ 全屏功能
- ✅ 视频封面图

### 3. 交互功能

- ✅ 点击推荐视频跳转到对应详情页
- ✅ 点击"查看更多"跳转到课程列表页
- ✅ 面包屑导航支持返回上级页面
- ✅ 路由参数变化时自动重新加载数据

### 4. 数据加载

- ✅ 调用API获取视频详情数据
- ✅ 调用API获取推荐视频列表
- ✅ 错误处理和加载状态
- ✅ 默认数据展示（API未配置时）

## API接口

### 获取详情

```typescript
/**
 * 获取实践案例详情
 * @param id 案例ID
 */
export function getExampleExpoDetail(id: string | number): Promise<ExampleExpoListItem>
```

**请求方式**: GET  
**接口地址**: `/xiThought/exampleExpo/detail/{id}`

### 获取推荐列表

```typescript
/**
 * 获取实践案例列表（分页）
 * @param params 查询参数
 */
export function getExampleExpoList(params: ExampleExpoListParams): Promise<ExampleExpoListResponse>
```

**请求方式**: POST  
**接口地址**: `/xiThought/exampleExpo/page/list`

## 使用方式

### 从首页跳转

在首页的学院案例模块中，点击任意课程卡片，会跳转到对应的详情页：

```typescript
// src/views/portal/Home/index.vue
const handleCourseClick = (course: Course) => {
  router.push(`/study/college-video/${course.id}`)
}
```

### 直接访问

通过URL直接访问：`http://localhost:5173/study/college-video/{id}`

## 设计规范

### 颜色

- 主题红色：`#BC2220`
- 基础黑色：`#333333`
- 背景色：`#FFF9ED`
- 边框色：`#FDD4A6`
- 渐变金色：`#BC7120` -> `#BC2220`

### 字体

- 字体家族：Source Han Sans CN (思源黑体)
- 标题字号：24px (font-weight: 500)
- 正文字号：16px (line-height: 1.75)
- 小字字号：14px

### 间距

- 卡片圆角：16px
- 卡片间距：24px
- 内边距：24px
- 按钮圆角：100px

## 响应式设计

- 桌面端（>1200px）：正常布局，推荐视频4列
- 平板端（992px-1200px）：推荐视频3列
- 移动端（<992px）：
  - 左右布局改为上下布局
  - 推荐视频2列或1列

## 注意事项

1. **视频格式**：确保视频文件格式为 MP4，编码为 H.264
2. **封面图**：建议尺寸 1920x1080，格式 JPG/PNG
3. **内容格式**：文字讲稿支持HTML格式，会自动添加段落缩进
4. **API配置**：确保后端接口已配置并返回正确的数据结构
